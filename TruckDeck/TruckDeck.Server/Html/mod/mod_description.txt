TruckDeck NAV
Author: TruckDeck
Version: 1.50
Release Date: July 2026

============================================================
DESCRIPTION
============================================================

TruckDeck NAV mirrors the TruckDeck turn-by-turn NAV dashboard
into your truck cabin as a transparent GPS / PC accessory display.
Designed for use with the free TruckDeck telemetry server on your
gaming PC (phone, tablet, or second monitor dashboards).

Website:  https://truckdeck.site
Downloads: https://truckdeck.site/downloads
Source:   https://github.com/dawood-shfuk/TruckDeck

FEATURES:
- Transparent in-cab GPS and PC displays (windshield corner or driver plate).
- Mirrors TruckDeck NAV from the TruckDeck server in your browser.
- Supports latest SCS trucks (2024-2026) and many community truck packs.
- Optional with TruckDeck — telemetry server, 9+ dashboards, input bridge.

============================================================
TRUCKDECK APP (COMPANION SOFTWARE)
============================================================

TruckDeck is a Windows telemetry server for ETS2 and ATS:

- Live REST telemetry API and browser dashboards on your LAN
- Trucker's Command Deck, OEM-style brand skins, classic Funbit skins
- Turn-by-turn NAV with offline maps (Map Generator)
- Native input bridge — map dashboard buttons to in-game keys
- Windows installer, Android APK, and this NAV cabin mod

Download TruckDeck-Setup.exe from https://truckdeck.site/downloads

============================================================
INSTALLATION
============================================================
1. Subscribe on Steam Workshop (or copy TruckDeck_NAV.scs to your mod folder).
2. Enable TruckDeck NAV in Mod Manager (priority above truck mods if needed).
3. Install GPS/PC via Cabin Accessories on supported trucks.
4. Run TruckDeck on your PC, open TruckDeck NAV in the browser, start driving.

Enjoy your drive with TruckDeck NAV!
