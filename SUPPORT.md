# Support TruckDeck development

TruckDeck is **free** for the sim racing community. It is maintained by **Dawood** as accessible trucking telemetry software — built so drivers can run dashboards on a phone, tablet, or second screen without expensive hardware.

If TruckDeck helps you on the road, consider supporting ongoing hosting, development, and accessibility work.

---

## Official site & downloads

| Resource | URL |
|----------|-----|
| **Home** | [https://truckdeck.site](https://truckdeck.site) |
| **Downloads** (installer, APK, NAV mod) | [https://truckdeck.site/downloads](https://truckdeck.site/downloads) |

The Windows installer, Android APK, and ETS2 NAV mod are also available from the in-app menu at `http://<your-pc-ip>:25555/downloads.html` on your local TruckDeck server.

## Other ways to help

- **Star** this repository on GitHub and share [truckdeck.site](https://truckdeck.site) with other drivers
- **Report bugs** via GitHub Issues (steps to reproduce, TruckDeck version, ETS2/ATS version)
- **Credit authors** when sharing screenshots or videos — see [CONTRIBUTORS.md](CONTRIBUTORS.md)
- **Contribute code** — pull requests welcome for skins, fixes, and documentation
